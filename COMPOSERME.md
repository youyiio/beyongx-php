# Composer 向导

## 阿里云镜像

> composer config repo.packagist composer https://mirrors.aliyun.com/composer/

## 腾讯云镜像

> composer config -g repos.packagist composer https://mirrors.cloud.tencent.com/composer/

## 重置为默认源

composer config -g --unset repos.packagist

## 安装

> composer install

> composer uupdate
